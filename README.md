# Vireon Interactive

  **Script Protection & Licensing System for Roblox**

  Built with React + Vite + Express + PostgreSQL.

  ## Features
  - 🛡️ Script Protection with HWID Locking
  - 🔑 Premium Key System (up to 3 years)
  - 🖥️ Admin Dashboard (code: APIP)
  - 👤 User Premium Dashboard
  - 🌐 Browser Detection (shows animated protection page)
  - 📨 Roblox notification format for expired/banned keys

  ## Tech Stack
  - Frontend: React 18 + Vite + Tailwind CSS (dark neon theme)
  - Backend: Express 5 (Vercel Serverless)
  - Database: PostgreSQL + Drizzle ORM
  - Monorepo: pnpm workspaces
